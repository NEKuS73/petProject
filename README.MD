# MyGoApp

MyGoApp is a simple Go application that processes financial transactions. It reads transaction data from a JSON file and calculates the total income, expenses, and remaining balance for a specified month.

## Features

- Reads financial data from a JSON file.
- Calculates total income and expenses.
- Displays the remaining balance or excess expenses.
- Supports multiple transaction categories.
